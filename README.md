# Counseling Center Operations Intelligence Platform

## Goal
Build an end-to-end analytics + ML pipeline to support operational decisions in a university counseling center.

## What this repo will include
- Data ingestion (real or synthetic)
- Cleaning + validation
- EDA + KPIs
- Forecasting / prediction (v1)
- Dashboard-ready outputs
- Clear documentation and reproducible runs

## Repo structure
- `src/` core code
- `notebooks/` experiments + EDA
- `data/raw/` raw inputs (not committed)
- `data/processed/` processed outputs (not committed)
- `docs/` decisions, logs, data dictionary
- `reports/` writeups
- `figures/` charts
